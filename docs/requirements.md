## Phase 3: 投稿閾値低下の工夫（差別化ポイント）

### 検討中の選択肢

1. **デイリーチェックイン** - 「今日の気分は？」など選択式＋一言で投稿完了
2. **お題/プロンプト投稿** - 日替わりお題を提示して埋めるだけ投稿
3. **ステータス更新型** - 「今なにしてる？」の一言＋絵文字だけで完了
4. **テンプレート投稿** - 定型フォーマットを選んで埋めるだけ
5. **スケジュールリマインダー** - 定期的にプッシュ通知で投稿を促す
6. **組み合わせ** - 上記の複数を組み合わせる

### 決定事項

以下の5つを実装する：

1. **目標宣言テンプレート**: 「今日の目標：___」をワンタップ投稿
2. **デイリーログ**: 「今日やったこと / 明日やること」を箇条書きで簡単記録
3. **応援リアクション**: いいねとは別に「応援！」ボタン
4. **週間振り返りプロンプト**: 定期的にお題を出す
5. **目標達成シェア**: 「目標を達成しました！」テンプレート

### 応援特典（将来検討）

応援ボタンを押しやすくするための特典案：

1. **応援バッジ/称号** - 応援数に応じてプロフィールにバッジ表示
2. **応援ランキング** - 週間/月間の応援数ランキング
3. **応援ストリーク** - 連続応援日数に応じてプロフィール表示
4. **応援された側も特典** - 応援された数に応じてプロフィールに表示
5. **応援レベル** - 応援数に応じてレベルが上がりプロフィールの見た目が変化
